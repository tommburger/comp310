Installing Ubuntu for Apple Silicon MacBooks

#2
General

  1. Download UTM from here.

  2. Download Ubuntu 22.04 from here.

  3. Open UTM and create a new VM.

  4. Press Open

  5. Select the Ubuntu 22.04.utm file that you just downloaded

  6. Launch the VM

  7. Once Ubuntu is launched, before logging in, press the power button on the top right corner, press 'Wired Unmanaged', and then press 'Connect'. It will ask you for a password. The password is 'ubuntu'. If you don't do this, the network may not work.

  8. Power Off the VM. It might switch to a black screen. Wait at that screen 1-2 minutes, and then fully close UTM and re-open the VM.

  9. Now login. The password for the ubuntu user is 'ubuntu'.

  10. Once logged in you should create a new user for yourself in Settings and then use that user in the future.

  11. Ubuntu should now be installed and working.

    For steps 7-9, you may also need to run sudo apt update in a terminal after you login to update apt’s mirrors for downloading things to work.
